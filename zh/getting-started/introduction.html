    # 鲁班H5是什么？

    鲁班H5是基于Vue2.0开发的，通过拖拽的形式，生成页面的工具，类似易企秀 (opens new window)、百度 H5 (opens new window) 等工具

    欢迎大家到鲁班H5-社区 (opens new window) 交流，在这里可以提问、反馈意见和建议，与作者直接互动
